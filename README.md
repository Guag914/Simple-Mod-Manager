# Simple Mod Manager

**Simple Mod Manager** is a lightweight utility that allows players to easily enable, disable, and manage their installed mods without manually moving files.

Designed to be fast, clean, and easy to use, it provides full control over your mod setup with minimal performance impact.

---

## Key Features
- Clean and intuitive in-game interface  
- Press **F8** to open the mod manager  
- Enable or disable mods with a single click  
- Automatically detects and organizes installed mods  
- Lightweight and optimized for performance  

---

## How It Works
1. Install Simple Mod Manager and launch the game  
2. Press **F8** at any time to open the manager  
3. Browse your installed mods  
4. Toggle mods quickly and easily  

---

## Why Use Simple Mod Manager?
Managing mods manually can be frustrating. Simple Mod Manager streamlines the process, making it easy to customize your setup without hassle.
